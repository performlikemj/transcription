# YardTalk Roadmap

This document outlines planned features and improvements for YardTalk.

## Current Version (v1.0)

- Basic dictation with Parakeet ASR model
- Global hotkey toggle (`Cmd+Shift+D`)
- Menu bar integration
- Floating waveform visualization
- Text insertion into active application

---

## Planned Features

### Customizable Hotkeys
- [ ] Settings UI to configure custom hotkey combinations
- [ ] Support for function keys (F1-F12)
- [ ] Support for media keys
- [ ] Multiple hotkey profiles
- [ ] Per-application hotkey overrides

### Enhanced Audio
- [ ] Microphone selection in settings
- [ ] Audio input level meter
- [ ] Noise suppression preprocessing
- [ ] Voice activity detection (VAD) for auto-stop

### Transcription Improvements
- [ ] Punctuation and capitalization options
- [ ] Custom vocabulary/words list
- [ ] Language selection (multi-language support)
- [ ] Real-time streaming transcription
- [ ] Confidence indicators

### User Interface
- [ ] Settings window with preferences
- [ ] Transcription history viewer
- [ ] Customizable waveform colors/styles
- [ ] Dark/light mode support
- [ ] Adjustable overlay position and size

### Text Processing
- [ ] Text commands (e.g., "new line", "period", "delete that")
- [ ] Text formatting commands (bold, italic)
- [ ] Clipboard mode (copy instead of type)
- [ ] Text replacement/correction macros

### Integration
- [ ] AppleScript support
- [ ] Shortcuts app integration
- [ ] Export transcriptions to file
- [ ] Cloud backup of settings

### Performance
- [ ] Faster model loading with caching
- [ ] Background model preloading
- [ ] Memory optimization for long sessions
- [ ] GPU acceleration improvements
